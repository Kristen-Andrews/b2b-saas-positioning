# Stress Test: Positioning Canvas Validation

Run your completed canvas against this checklist before locking it. If any item fails, go back to that section.

## Category (Section 1)

- [ ] Category name is 4 words or fewer
- [ ] Category name uses title case
- [ ] Category name is not a feature description
- [ ] A competitor cannot claim the same category name without sounding forced
- [ ] The one-sentence category definition fits in a tweet

## ICP (Section 2)

- [ ] A rep can name 3 accounts in their territory that match exactly
- [ ] The trigger event is a specific event, not a persona description
- [ ] The champion buyer has a specific title, not a range of titles
- [ ] The economic buyer has board or CEO exposure
- [ ] The ICP excludes at least 60% of the total addressable market

## Primary alternative (Section 3)

- [ ] The primary alternative is NOT a direct competitor for most ICP accounts
- [ ] You can describe the current workflow in the customer's words
- [ ] The reason it persists is specific (not "inertia" or "budget")
- [ ] You can name the specific failure mode that pushes them to look

## Unique attributes (Section 4)

- [ ] Every attribute passes the swap test with your top competitor
- [ ] No attribute uses generic descriptors ("AI-powered", "enterprise-grade", "purpose-built")
- [ ] At least 3 attributes, no more than 5
- [ ] Each attribute is a fact, not an adjective

## Value proof (Section 5)

- [ ] Every attribute has exactly one proof point
- [ ] At least 2 proof points are named customer + specific metric
- [ ] No proof point requires a paragraph to explain
- [ ] Every proof point is verifiable (customer would confirm, or documentation exists)

## Enemy (Section 6)

- [ ] The enemy is a condition, not a competitor
- [ ] Your best customer would nod at the enemy statement without you in the room
- [ ] The cost of staying stuck is quantifiable
- [ ] The "new way" is a category-level statement, not a feature

## Message map (Section 7)

- [ ] Headline is 7 words or fewer
- [ ] Subhead names the ICP, the outcome, and one differentiator
- [ ] Each pillar has a claim and a proof point
- [ ] Elevator pitch runs 30 seconds or less when spoken
- [ ] Boilerplate fits in two sentences

## Anti-positioning (Section 8)

- [ ] At least 3 "not for" statements
- [ ] At least 3 "not doing" statements
- [ ] The "when we lose" statement is honest, not defensive
- [ ] Sales leadership signed off on the anti-positioning

## Overall integrity

- [ ] Every section ties back to the category defined in Section 1
- [ ] The message map (Section 7) uses language from the ICP (Section 2), not internal jargon
- [ ] The unique attributes (Section 4) map to the proof points (Section 5) one-for-one
- [ ] The enemy (Section 6) explains why the primary alternative (Section 3) is broken
- [ ] Nothing in the canvas requires a competitor comparison table to make sense

## Final gate

- [ ] Product leadership signed off
- [ ] Sales leadership signed off
- [ ] CEO signed off
- [ ] Marketing team can use this canvas to reject copy, campaign concepts, and PR angles that don't fit

If you can't check the final gate, you don't have positioning. You have a document.
